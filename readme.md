> Nothing


## Install

```
$ npm install --global alfred-staerkio
```

*Requires [Node.js](https://nodejs.org) 4+ and the Alfred [Powerpack](https://www.alfredapp.com/powerpack/).*


## Usage

In Alfred, type `sio`, <kbd>Enter</kbd>, and your query.
